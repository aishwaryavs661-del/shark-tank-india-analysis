# 🦈 Shark Tank India — Who Gets the Money?

> *117 startups walked in. Only 65 walked out with a deal.*

## 🎯 What This Project Is About
A deep-dive into Shark Tank India's investment data — uncovering which 
industries the sharks love, who invests the most, and what separates 
a funded startup from a rejected one.

---

## 💥 The Big Numbers

| Metric | Value |
|--------|-------|
| 🏢 Startups Analyzed | 117 |
| 🤝 Deals Closed | 65 |
| ❌ Sent Home Empty | 52 |
| 📈 Success Rate | **55.6%** |
| 💰 Total Invested | **₹36.42 Crore** |
| 🏆 Highest Funded | **Aas Vidyalaya (₹150 Lakhs)** |

## 🏭 Which Industry Wins?

🥈 Health & Wellness — **66.7%**  
🥉 Technology — **61.9%**
